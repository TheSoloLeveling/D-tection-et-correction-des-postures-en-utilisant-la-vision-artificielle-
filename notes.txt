Dynamic Time Warping est un algorithme utilisé pour synchroniser deux séries temporelles ou deux sequences.
la methode naive de comparaison est de calculer la distance entre chaque point puis effectuer la moyenne

par exemple K-nearest neighbours
on compare deux items qui ont les mêmes dimensions on peut utiliser distance euclidienne si ce n'est pas le cas
on peut utilise Dynamic Time Warping comme une distance metric si les données sont des signals ou des sequences de differentes
longueurs.

l'algorithme aligne les points entre les deux signals, on peut donc savoir quels points correspondent les uns aux autres et par suite
il donne un cost d'alignement qui vas permettre de savoir à quel point les deux signaux sont-ils similaires
